# SquidDeck
A smart circular Deck powered by an Arduino Nano. Tapping an NFC-enabled phone triggers colorful animations on a 24-LED WS2812B ring. Powered by a LiPo battery with onboard charging (TP4056), and using the NXP NT3H2111 NFC chip with a custom antenna.
